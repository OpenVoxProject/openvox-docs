Warning: 'puppet man' is deprecated and will be removed in a future release.

USAGE: puppet man *action*

Please use the command 'puppet help *subcommand*' or the system manpage system 'man puppet-*subcommand*' to display information about Puppet subcommands. The deprecated man subcommand displays manual pages for all Puppet subcommands. OPTIONS: --render-as FORMAT - The rendering format to use. --verbose - Whether to log verbosely. --debug - Whether to log debug information.

ACTIONS: man Display the manual page for a Puppet subcommand.

See 'puppet help man' or 'man puppet-man' for full help.
