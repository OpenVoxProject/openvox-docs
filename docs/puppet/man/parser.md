USAGE: puppet parser *action*

Interact directly with the parser.

OPTIONS: --render-as FORMAT - The rendering format to use. --verbose - Whether to log verbosely. --debug - Whether to log debug information.

ACTIONS: dump Outputs a dump of the internal parse tree for debugging validate Validate the syntax of one or more Puppet manifests.

See 'puppet help parser' or 'man puppet-parser' for full help.
