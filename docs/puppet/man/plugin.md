---


USAGE: puppet plugin *action*

This subcommand provides network access to the puppet master's store of plugins.

OPTIONS: --render-as FORMAT - The rendering format to use. --verbose - Whether to log verbosely. --debug - Whether to log debug information.

ACTIONS: download Download plugins from the puppet master.

See 'puppet help plugin' or 'man puppet-plugin' for full help.
